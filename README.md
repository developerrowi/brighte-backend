## Tech Specification

- NodeJS  v20.18.3

## Manual Installation

- `git@github.com:developerrowi/brighte-backend.git`
- `cd brighte-backend/`
-  add or update `.env` variables in file directory
- `npm install`

## Run using docker 
- `docker-compose build && docker-compose up`

## Dev Build 
-  `npm run dev`

## Prod Build
-  `npm run build && npm run start`
